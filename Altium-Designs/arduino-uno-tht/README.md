# Arduino Uno THT Version

This project is a through-hole (THT) version of the Arduino Uno board designed for easier manual assembly and prototyping.

> **Note:** This board is intended as a hardware replica. The ATmega328P microcontroller must be pre-programmed using an external Arduino board or bootloader. It is **not possible to upload code directly to this board**.

## Features

- ATmega328P microcontroller (pre-programmed)
- Arduino Uno compatible pinout
- Through-hole components for easy soldering
- USB to Serial interface
- 5V voltage regulation

## Design Goals

- Learn microcontroller board architecture
- Practice PCB routing and component placement
- Understand power and ground layout
- Develop hands-on skills in assembling through-hole boards
- Provide a functional Arduino-compatible board without onboard programming capability

## Tools Used

- Altium Designer
- ATmega328P datasheet
- Arduino Uno reference schematic
